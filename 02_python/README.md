# 動作確認

## 実行方法

IDEの実行ボタンを押す

## 確認方法

### task01.ipynb

#### 期待する結果

```text
変数に設定した名前と年齢を使って自己紹介文が表示される。
来年の年齢は age + 1 で計算される。
```

#### 実行例

| 入力値 | 出力                                                                                                           |
| :----- | :------------------------------------------------------------------------------------------------------------- |
| なし   | はじめまして、 テスト太郎 と申します。<br>年齢は 20 歳です。来年は 21 歳になります。<br>よろしくお願いします。 |

### task02.ipynb

#### 期待する結果

```text
入力された税抜価格に消費税 10% を加算し、
税込価格が 2000 円以上なら送料無料、未満なら送料 350 円を加算する。
送料と送料込税込価格が表示される。
```

#### 実行例

| 入力値 | 出力                                          |
| :----- | :-------------------------------------------- |
| 2000   | 送料は無料です<br>送料込税込価格は2200円です  |
| 1500   | 送料は350円です<br>送料込税込価格は2000円です |

### task03.ipynb

#### 期待する結果

```text
入力された元金と年利をもとに、
10年間の複利計算を行い、10年後の金額を表示する。
```

#### 実行例

| 入力値                  | 出力                  |
| :---------------------- | :-------------------- |
| 元金 = 1000<br>年利 = 5 | 10年後の金額: 1628.89 |

### task04.ipynb

#### 期待する結果

```text
2つの数値を入力し、割り算を行う。
正常時は計算結果を表示する。
文字入力時はエラーメッセージを表示する。
0で割る場合もエラーメッセージを表示する。
```

#### 実行例

| 入力値  | 出力                            |
| :------ | :------------------------------ |
| 100, 10 | 計算結果: 10.0                  |
| 100, 0  | エラー: 0で割ることはできません |
| a, 10   | エラー: 数値を入力してください  |
| 100, b  | エラー: 数値を入力してください  |
