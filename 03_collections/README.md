# 動作確認

## 実行方法

IDEの実行ボタンを押す

## 確認方法

### task01.ipynb

#### 期待する結果

```text
学生の成績リスト（grades）をもとに平均点を計算して表示する。
最高点と最低点を求めて表示する。
平均以上の成績の人数をカウントして表示する。
```

#### 実行例1

| 入力値                        | 出力                                                            |
| :---------------------------- | :-------------------------------------------------------------- |
| grades = [85, 90, 78, 92, 88] | 平均点: 86.6<br>最高点: 92<br>最低点: 78<br>平均以上の学生数: 3 |

#### 実行例2

| 入力値                        | 出力                                                            |
| :---------------------------- | :-------------------------------------------------------------- |
| grades = [70, 75, 80, 85, 90] | 平均点: 80.0<br>最高点: 90<br>最低点: 70<br>平均以上の学生数: 3 |

### todo.ipynb

#### 期待する結果

```text
起動時に todo_list.txt が存在すれば読み込み、存在しなければ空のリストで開始する。
番号入力で操作できる。
1を入力するとTODOを追加できる。
2を入力するとTODOを番号付きで表示できる。
3を入力するとTODOを番号付きで表示し、指定した番号のTODOを削除できる。
指定した番号が無効な場合や、数字以外が入力された場合は「無効な番号です。」と表示し、削除操作がキャンセルされる。
4を入力するとTODOを todo_list.txt に保存して終了する。
```

#### 実行例（3:削除 のあとで、正しい番号（2）を入力した場合）

```text
【TODOリスト】
番号を入力してください[1:追加、2:表示、3:削除、4:保存と終了]:3
3:削除
1. 英語の勉強
2. 切手を買う
削除する番号: 1
番号を入力してください[1:追加、2:表示、3:削除、4:保存と終了]:2
2:表示
1. 英語の勉強
番号を入力してください[1:追加、2:表示、3:削除、4:保存と終了]:4
4:保存と終了
TODOリストを保存して終了します。
```

2:表示
