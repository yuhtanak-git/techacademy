# 動作確認

## 実行方法

IDEの実行ボタンを押す

### task01.ipynb

#### 期待する結果

```text
ユーザー名は5文字以上15文字以内であること。
ユーザー名に空白を含まないこと。
ユーザー名は英数字（アルファベットと数字）のみであること。
条件を満たす場合は「ユーザー名は有効です」と表示する。
条件を満たさない場合はエラーメッセージを表示し、その後「不正なユーザー名です」と表示する。
```

#### 実行例

| 入力値     | 出力                                                                              |
| :--------- | :-------------------------------------------------------------------------------- |
| JohnDoe123 | ユーザー名は有効です                                                              |
| JD         | エラー: ユーザー名は5文字以上15文字以内で入力してください<br>不正なユーザー名です |
| John@Doe   | エラー: ユーザー名は英数字のみ使用できます<br>不正なユーザー名です                |
| John Doe   | エラー: ユーザー名に空白を含めることはできません<br>不正なユーザー名です          |

### 期待する結果

```text
指定したURLからデータを取得し、UTF-8形式でデコードしてから先頭500文字を取得する。

```
